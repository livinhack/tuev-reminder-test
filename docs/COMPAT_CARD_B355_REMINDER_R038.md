# Compatibility – Card b355+ / b356 RC with Reminder r038

Reminder r038 remains compatible with the existing Card bridge attributes from b355+ / b356 RC.

The r038 change adds a Reminder-owned Manager create backend path and a small Sidebar UI plus-control refinement. It does not change sensor attribute names, Card-facing bridge fields, Card rendering, Card actions, calendar event semantics or the existing Dashboard card contract.
