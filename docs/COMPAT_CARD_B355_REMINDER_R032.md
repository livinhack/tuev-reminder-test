# Compatibility – Card b355 + Reminder r032

Reminder r032 keeps Card compatibility unchanged.

## Compatible baseline

- Card: b355 or newer b356 RC
- Reminder: r032

## r032-specific note

r032 only improves the Reminder-owned Sidebar read-only vehicle list. It does not change sensor attributes, Card bridge fields, Card actions, plate rendering or Dashboard behavior.

The Card remains a separate Lovelace/Dashboard project and is not bundled into the Reminder repository.
