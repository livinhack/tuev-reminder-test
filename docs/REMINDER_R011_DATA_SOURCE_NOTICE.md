# Reminder r011 – Data Source Notice

r011 bundles a local area-code suggestion list for German license plates.

Source:

```text
openpotato/kfz-kennzeichen
src/de/kennzeichen.csv
```

The repository documents the data source as KBA / German legal sources and is licensed under ODbL-1.0. r011 includes the ODbL license text next to the bundled data.

The TÜV Reminder uses this database only as a suggestion aid. It does not decide whether a plate area code may currently be assigned.
