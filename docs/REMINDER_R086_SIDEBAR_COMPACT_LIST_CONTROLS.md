# Reminder r086 – Sidebar Compact List Controls

r086 continues function/UI development after r085. It consolidates the Sidebar list controls into a single compact band and keeps release work parked.

## Changes

- Search and status chips now share one `.list-controls` area.
- The search clear `×` stays inside the search input.
- Status reset remains the `Alle` chip.
- The hit counter is reduced to the visible/total count.
- The verbose technical Sidebar capability detail is hidden from the normal UI.

## Preserved

- Card b355 bridge compatibility.
- `calendar.tuev_reminder` detached calendar behavior.
- `reminder_offset_days` behavior.
- Existing Sidebar CRUD, validation, duplicate checks, dirty guard and mobile action sheet.
- No Card files and no release steps.
