# Compatibility – Card b355+ / b356 RC with Reminder r042

Reminder r042 remains Card-compatible with the established b355+ attribute bridge.

The Sidebar changes are Reminder-owned manager UI changes only. The Card remains a separate Dashboard/Lovelace repository and is not imported or bundled.

## Runtime notes

- Existing sensor attributes remain unchanged.
- The create flow from r041 remains available.
- The new three-dot action menu is UI-only for now; no update/delete backend command is added in this version.
