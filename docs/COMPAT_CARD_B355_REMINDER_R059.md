# Compatibility – Card b355 with Reminder r059

r059 changes only Sidebar form input constraints and local validation parity. Manager API payloads, sensor attributes, calendar behavior, services, Card bridge fields, panel registration, responsive table rules, brand asset paths, and CRUD semantics are unchanged from r058.

The Lovelace/Dashboard Card remains a separate project. Reminder continues to own the data/ConfigEntries; the Card only reads the resulting entities and attributes.
