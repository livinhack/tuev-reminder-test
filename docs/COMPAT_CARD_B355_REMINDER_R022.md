# Compatibility – Card b355 + Reminder r022

Reminder r022 is compatible with Card b355.

r022 preserves the runtime behavior from the stabilized v3 line:

- Card bridge attributes remain available.
- Plate whitespace is preserved.
- H/E boolean suffix attributes remain available.
- Green, seasonal, green + seasonal, change plate and plate format attributes remain available.
- `calendar.tuev_reminder` remains detached from individual vehicle devices.
- Calendar always emits reminder + due events.
- `reminder_offset_days` remains available.

No Card update is required for r022.

Reminder r023 keeps this compatibility line active for Card b355.

Reminder r024 preserves this compatibility baseline for Card b355 + Reminder r024.
