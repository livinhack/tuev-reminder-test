# Compatibility · Card b355+/b356 RC with Reminder r033

Reminder r033 does not change the sensor attributes or Manager API read model used by the Card.

The Reminder Sidebar panel remains separate from the Card repository. The new row-end plate preview is a lightweight Reminder UI fallback only and does not import, bundle or mutate Card renderer code.

Expected compatibility remains: Card b355+ / b356 RC with Reminder r033.
