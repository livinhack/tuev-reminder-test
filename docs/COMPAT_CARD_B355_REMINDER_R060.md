# Compatibility – Card b355 with Reminder r060

r060 changes only the Sidebar form UI for Manager create/edit flows. The Sensor/Card bridge attributes, Manager API payload fields, services, calendar behavior, brand assets, and CRUD semantics are unchanged from r059.

The Lovelace/Dashboard Card remains a separate project. Reminder r060 still provides the same Card-facing attributes and does not import Card code or duplicate Card actions.

Card bridge fields remain unchanged for Card b355 compatibility.
