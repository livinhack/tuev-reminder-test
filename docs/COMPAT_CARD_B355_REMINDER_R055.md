# Compatibility – Card b355+ with Reminder r055

Reminder r055 only changes Sidebar row-action event handling for smartphone action sheets.

No Card bridge fields changed. Existing Dashboard Cards continue to read the same Reminder sensor attributes.

Preserved bridge attributes include:

```text
plate_suffix_h
plate_suffix_e
plate_color_mode
seasonal
season_start_month
season_end_month
change_plate_enabled
change_plate_common_text
change_plate_vehicle_text
change_plate_vehicle_digit
```
