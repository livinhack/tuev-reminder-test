# Reminder r104 – Sidebar Form Section Invalid Feedback

r104 builds on r102 and keeps the accepted r100/r097 layout. Invalid fields are still marked directly, but the surrounding form card now receives subtle invalid feedback as well. This makes it easier to see which part of the form needs attention without adding new helper rows or changing the form structure.

Unchanged: the right overview card, the separate Saison card below it, the compact dark Kennzeichen fallback, and all Card-renderer/Card-detection boundaries.
