# Compatibility – Card b355+ with Reminder r049

Reminder r049 remains compatible with the existing Card bridge attributes.

The r049 change is Sidebar-only UX hardening around unsaved changes. It does not change sensor attributes, Card-facing data names, service behavior, or the separation between the Reminder integration repository and the dashboard Card repository.

No Card files are copied into the Reminder repository and no Card renderer/API is imported.
