# Compatibility – Card b355+ with Reminder r047

Reminder r047 remains compatible with the existing Card bridge attributes.

The Sidebar changes are limited to Reminder-owned manager UI behavior:

- row click behavior removed;
- Drei-Punkte-Menü remains the only per-row action entry point;
- sortable table headers were added;
- create/update API from r041/r045 remains unchanged.

No Card files, renderer internals, Lovelace configuration, or Card actions are copied into the Reminder repository.
