# Compatibility – Card b355 + Reminder r086

Reminder r086 remains compatible with the existing Card b355 bridge line. This step only changes Sidebar Manager layout.

## Unchanged integration behavior

- Vehicle ConfigEntry/entity behavior is unchanged.
- The detached calendar entity remains `calendar.tuev_reminder`.
- `reminder_offset_days` remains the reminder offset field.
- The Card-facing data model is not changed.

## UI-only scope

- Search and status quick filters are visually consolidated.
- The normal list view no longer shows the verbose technical Sidebar capability line.
- No Card files are changed.
