# Reminder r077 – Sidebar List Controls Bundle

r077 is a bundled UI/UX development step for the Reminder Sidebar Manager. It intentionally combines several small list-control improvements into one ZIP to reduce intermediate artifacts.

## Scope

- Adds status summary chips for quick filtering.
- Keeps existing search, status select and sortable headers.
- Adds sticky table headers and stronger status-pill scan cues.
- Keeps Create/Edit/Delete runtime behavior unchanged.
- Keeps Reminder/Card separation unchanged.

## Not included

- No release-candidate step.
- No public release asset.
- No Card renderer import.
- No Lovelace/Card configuration logic.
